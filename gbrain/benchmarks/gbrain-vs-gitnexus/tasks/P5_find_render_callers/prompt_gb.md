# Task: Find All Callers of render()

Find all callers of Preact's render() function.

## Your Task
1. Find where render() is defined in Preact source
2. List all functions that directly call render()
3. Write render_callers.md with each caller in format: FuncName (file:line)

## Expected Outcome
A markdown file listing all callers with file:line references.

## MANDATORY: Using GBrain Code Tools
1. code_query "render" to find the symbol
2. code_context "render" to get the full call graph
Base your answer on the code_context output.
